# ASAPP Example Project

Created using react [react hot boilerplate](https://github.com/gaearon/react-hot-boilerplate)

## Installation
### Intall the following
- nvm (or just use the correct version of node specified in `.nvmrc`)
- yarn (or npm)

```bash
nvm use
yarn
```
## Development
To start:    
```bash
npm start
```
To test:    
```
npm t -s
```
